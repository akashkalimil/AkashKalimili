Intertial Navigation System with Wireless Communication and Data Processing with a FPGA

Using an accelerometer as a source of data, I designed and built analog signal conditioning circuitry to optimize the signal for digitization and serialization. After building ADC circuitry and Serialization Circuitry, I ran many diagnostic tests for baud rate and data loss with oscilloscopes to make sure I wasn't over sampling or undersampling. I used an RN-41 Bluetooth module. I used a microcontroller (Ardiuno) to program program protocols on the flash memory of the chip. On the recieving end of the connection, an FPGA (Nexys 5) was hooked up to the reciever. Using Xilinx , I implemented integration algorithms that I created. My solution for integration drift was numerous trials to calculate constants. At the end of the year, I presented this project at the annual TJStar Convention (Large event where research projects are presented to the school community.
